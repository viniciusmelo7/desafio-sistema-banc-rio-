# 🏦 Sistema Bancário em Python

Projeto desenvolvido em Python com foco em lógica de programação, programação orientada a objetos e simulação de operações bancárias em ambiente de linha de comando.

O sistema permite realizar operações básicas de gerenciamento bancário, incluindo cadastro de usuários, criação de contas, depósitos, saques e consulta de extrato.

---

# 🚀 Funcionalidades

✅ Cadastro de usuários
✅ Criação de contas bancárias
✅ Depósito em conta
✅ Saque com validações
✅ Consulta de extrato
✅ Listagem de contas cadastradas
✅ Controle de limite de saque
✅ Controle de quantidade de saques diários

---

# 🛠 Tecnologias Utilizadas

* Python 3
* Programação Orientada a Objetos (POO)
* Estruturas condicionais e de repetição
* Manipulação de listas e dicionários
* Interface via terminal

---

# 📋 Regras do Sistema

O sistema possui algumas regras de negócio, como:

* Limite máximo de saque de R$ 500,00 por operação
* Máximo de 3 saques diários
* Validação de saldo disponível
* Controle de movimentações através do extrato

# ▶️ Como Executar

## Pré-requisitos

* Python 3.6 ou superior instalado

## Executando o projeto

Clone o repositório:

```bash
git clone <url-do-repositorio>
```

Acesse a pasta do projeto:

```bash
cd sistema-bancario
```

Execute o arquivo principal:

```bash
python sistema_bancario.py
```
